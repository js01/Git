配置用户名和邮箱 ​

在以后提交到仓库的时候需要指定用户名和邮箱，这里提前进行全局设置，以后默认使用该身份。使用如下命令设置用户名和邮箱：

sh
git config --global user.name "姓名"
git config --global user.email "邮箱"
